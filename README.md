## 🚀 Getting Started

### Prerequisites

Ensure you have the following installed:

```bash
# C++
g++ --version          # GCC 9.0+

# C
gcc --version          # GCC 9.0+

# Java
java -version          # JDK 11+
javac -version         # Compiler required
```

### Installation

1. **Clone the repository**

```bash
git clone https://github.com/shriram7057/Unstop-Problem-Solving-100-Days.git
cd Unstop-Problem-Solving-100-Days
```

2. **Navigate to your preferred language**

```bash
cd C++              # or C/ or Java/
```

---

## 💻 Usage

### Running C++ Solutions

```bash
cd C++/001-050
g++ -std=c++17 -O2 problem_01.cpp -o problem_01
./problem_01
```

### Running C Solutions

```bash
cd C/001-050
gcc -O2 problem_01.c -o problem_01
./problem_01
```

### Running Java Solutions

```bash
cd Java/001-050
javac Problem01.java
java Problem01
```

## 📝 Best Practices

When solving problems:

```
1. Understand the problem statement clearly
2. Identify patterns and constraints
3. Plan your approach before coding
4. Write clean, well-commented code
5. Test with edge cases
6. Analyze time and space complexity
7. Optimize if necessary
```
